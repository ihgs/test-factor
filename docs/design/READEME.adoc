== イメージ

以下のファイルを読み込むと、

```
[factor]
----
* A: userId
  1. 正しい値
  2- 入力なし
  3- 100文字
* B: password
  1. 正しい値
  2- 入力なし
  3- 間違った値
----

[combination]
----
A:1,B:1
A:1,B:[2,3]
A:[2,3],B:1
----
```
